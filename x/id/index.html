# Id

The id module is the one that allows you to create a new identity and associate to it a Did Document.

# Transactions

Using the id module you can perform the following transactions.

Accessible to everyone

  • Create a new identity
  • Associate a Did Document to an identity
  • Request a Did deposit
  • Cancel a Did deposit request
  • Request a pairwise Did power up
  • Cancel a pairwise Did power up request

Accessible to the government

  • Reject a Did deposit request
  • Reject a pairwise Did power up request
  • Accept a deposit request
  • Power up a Did
