# Mint

The mint module is the one that allows you to create Collateralized Debt Positions (CDPs) using your Commercio.network tokens (ucommercio) in order to get Commercio Cash Credits (uccc) in return.

# Transactions

Using the mint module you can perform the following transactions.

  • Opening a CDP
  • Closing a CDP

# Queries

  • Read a CDP by its owner's address and timestamp
  • List all CDPs of a user
