# Pricefeed

The pricefeed module allows to external actors (called oracles) to insert into the Commercio.network blockchain the current prices of different assets such as tokens and NFTs.

# Transactions

Using the pricefeed module you can perform the following transaction.

Accessible only to the oracles

  • Set a raw price for a given asset

Accessible only to the government

  • Adding a new oracle
